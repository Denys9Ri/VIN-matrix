Політика конфіденційності

1. Хто обробляє дані

2. Які дані можуть оброблятися

  • дані облікового запису: логін, ім’я, номер телефону та інша інформація, яку користувач самостійно додає;
  • дані компанії та працівників, необхідні для налаштування робочого простору;
  • дані клієнтів, автомобілів, візитів, товарів, документів і платежів, які користувач вносить до системи;
  • технічні дані, потрібні для стабільності й безпеки сервісу: журнали помилок, IP-адреса, відомості про пристрій і браузер.

3. Для чого використовуються дані

  • для створення та підтримки облікового запису й робочого простору;
  • для надання функцій VIN-matrix: CRM, обліку, документів, аналітики та інших модулів;
  • для технічної підтримки, захисту від зловживань і виправлення помилок;
  • для виконання юридичних обов’язків, коли це вимагає законодавство.

4. Дані клієнтів користувача

Користувач самостійно визначає, які дані його клієнтів і працівників вносяться до VIN-matrix, та відповідає за наявність належних підстав для такої обробки. RDmatrix Company використовує ці дані виключно для надання й підтримки функціональності сервісу.

6. Передавання третім особам

Дані можуть передаватися технічним підрядникам лише в обсязі, необхідному для роботи сервісу, або у випадках, прямо передбачених законодавством. Ми не продаємо персональні дані користувачів.

Для уточнення, оновлення чи видалення даних зверніться до підтримки за номером +38 066 688 14 91. За потреби ми підкажемо безпечний спосіб підтвердити особу та оформити запит.
